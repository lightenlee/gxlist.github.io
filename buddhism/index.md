# 佛教

[南无第三世多杰羌佛](djc)

十方诸佛共有不二的行持教戒：『远离封建迷信、怪力乱神、邪师骗子、附佛外道、邪教、邪书。要严持佛教戒律，大悲为本，诸恶莫作，众善奉行，舍己利他，忍辱愧行，自净其意，面对众生不分残缺病康，一律平等视为亲人，要知万法皆因果，善因得善报，善报结善果，善果获正法，依法圆福慧，步入成就境，脱离众生苦，了脱生死轮，圆满成佛道！！！』
